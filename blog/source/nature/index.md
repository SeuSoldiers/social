# 殿镇村周边的自然风光
殿镇村位于陕西省西安市周至县集贤镇，因其优美的自然风光而闻名。村庄四周群山环抱，植被茂密，空气清新，宛如一幅天然的山水画卷。清晨，薄雾笼罩，远山若隐若现，仿佛仙境；傍晚，夕阳西下，晚霞映照，整个村庄沐浴在金色的余晖中，景色迷人。村内溪流潺潺，鸟语花香，田园风光与自然景观相得益彰，令人流连忘返。此外，当地农家乐众多，村民们热情好客，为游客提供了舒适的休憩之所。
